Automations MCP server: run workflows directly from Claude

The platform exposes an MCP server at api.implementations.goldenweb.net/mcp/mcp. Claude Code connects via a personal API key and runs the full loop: analyse, review, approve, apply to Shopify.

  • Browse available SEO workflows and categories
  • Start analysis with inline params or staged data
  • Block until analysis completes, returns proposal summary
  • Review generated proposals with status filters
  • Bulk-approve all pending proposals
  • Enqueue approved proposals for Shopify write
  • Track execution progress (applied / failed / skipped)
  • Start meta title + description workflows for a single URL
  • Session re-entry: see what completed while away
Shopify MCP in Claude Web: new capabilities unlocked

Claude Web now has direct access to the Shopify MCP toolset, enabling full operational control across catalog, commerce, content, bulk pipelines, and validation flows.

  • Store and catalog intelligence: shop info, markets, price lists, blogs, articles, and tag operations
  • Product operations: create, update, delete products; bulk variant create/update/delete; detailed variant inspection
  • Customer operations: search and paginate customers; create/update/delete; merge records; manage addresses
  • Order operations: query and inspect orders, update metadata, create/complete draft orders, cancel/close/reopen, mark paid, fulfill, refund
  • Inventory operations: list locations, inspect inventory items and per-location levels, set on-hand quantities
  • Metafield operations: read, create/update, delete metafields, and inspect metafield definitions by owner type
  • Generic resource access: list resource types, query sampled datasets, fetch resources by ID or handle
  • GraphQL access: execute read-only Shopify GraphQL queries with mutation blocking for safety
  • Bulk data pipelines: start export/import jobs, poll status, list active jobs, and cancel running jobs
  • Theme workspace bootstrap: enumerate themes, prepare duplication/rename mutation templates, and generate workspace setup files
  • URL validation at scale: stage large URL sets, run async validation, poll job status, and export results

CrewAI agents for title and description writing with self-review

Two AI agents run in sequence: a Writer generates candidates, a Reviewer scores them on accuracy and SEO fit. If nothing passes, the critique goes back to the Writer for another attempt.

Security Incident: March 2026

On 1 March 2026 the server was compromised via CVE-2026-24841, a command injection flaw in Dokploy v0.25.6. The attacker got shell access inside the Next.js container and tried to install a cryptocurrency miner, which failed. No host compromise, no data exfiltration, no persistence. Infrastructure has been hardened and rebuilt.

Actions taken & current security posture
01Dokploy updated to v0.28.2, patches CVE-2026-24841
02All secrets rotated: Clerk keys, DB passwords, Shopify tokens, JWT secrets
03SSH password authentication disabled, key-only enforced
04Both containers rebuilt from clean images — full malware scan passed
05Traefik access logging enabled with 30-day rotation for future forensics
06CrowdSec active — 349 attacker IPs banned, 106,450 SSH attempts blocked
